Для запуска теста нужен только Node.js

После клонирования проекта запустить команду обновления пакетов
npm update

и можно запускать тест
node test.js

во время прохождения теста по очереди используется сразу два движка: Webkit и chromium
После проверки создаются скриншоты в качестве бонуса
так же в коде можно выставить нужное разрешение вьюпорта
